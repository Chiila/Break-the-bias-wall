# Break the Bias Wall

Teams treat bias as a “bug” by identifying biased statements in a “corrupted code” paragraph using a code-editor-style setup. They highlight the issues and rewrite them into fair, inclusive statements, then enter the wall passphrase to finish—symbolizing the removal of bias.

Built with [Next.js](https://nextjs.org).

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
